# ayx-panel-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，作为静态资源集合存在。页面内容不针对任何单个域名或具体网站，主要用于存放通用的前端资源与示例页面。

## 目录说明

```
├── pages/          # 存放独立的 HTML 页面文件
├── assets/         # 页面引用的样式、脚本、图片等资源
├── README.md       # 本文件
└── LICENSE         # 许可证文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，文件名可反映其用途。
- `assets/`：按需组织子目录，如 `css/`、`js/`、`images/`。

## 页面归档说明

- 所有页面均为独立可运行的 HTML 文件（包含内联或外部引用资源）。
- 页面内容保持通用性，不绑定特定域名或服务。
- 归档页面可能包含示例代码、演示界面或工具型页面。
- 部分页面可能需要联网加载外部资源（如字体、库文件），建议在对应页面内注明依赖。

## 维护说明

- 欢迎提交 PR 添加新的通用页面，请确保内容合规且不涉及侵权。
- 页面文件命名建议使用英文小写字母、数字与连字符，避免特殊字符。
- 如有资源更新，请同步修改对应页面引用路径。
- 本仓库仅作为资源存储与版本控制使用，不提供在线演示或托管服务。
